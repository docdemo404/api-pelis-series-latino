🎬 API Películas & Series Latino

Documentación oficial humana y machine-readable para desarrolladores y Agentes de IA (LLMs).

🤖 Especificación OpenAPI 3.0 (JSON)
✨ Catálogo Unificado & Gestor de Fuentes: La API agrupa automáticamente títulos idénticos entre diferentes proveedores (TioPlus, FuegoCine, Supabase) unificando sus servidores. Puedes gestionar la activación y prioridad de fuentes en /panel o /api/v1/panel.
🎬 TMDB API Enriquecida: Integración con la API oficial v3 de TheMovieDatabase (TMDB). Todos los contenidos devuelven sinopsis oficiales en Latino, tráilers oficiales en YouTube, pósters/backdrops en alta definición, reparto con fotos y calificaciones.
🤖 Machine-Readable API & Alta Velocidad (Edge CDN & In-Memory Cache): Respuestas ultrarrápidas de sub-20ms con Edge Caching (Cache-Control: public, s-maxage=86400, stale-while-revalidate), soporte de batching (/api/v1/media/batch?ids=...), paginación real sobre el índice general (?page=2&limit=25 o ?page=2&size=25) y proyecciones compactas (?compact=true). Especificación OpenAPI 3.0 disponible en /api/v1/openapi.json.
⚠️ Regla de Consumo de Servidores: El campo status de cada servidor en servers indica su disponibilidad real comprobada en capa de aplicación (inspección de Soft Errors). Todo reproductor o aplicación cliente DEBE IGNORAR O FILTRAR los servidores que retornen "status": "offline".

📌 Endpoints Principales

GET / POST /panel | /api/v1/panel

Panel de Gestión de Fuentes: Interfaz interactiva y endpoint JSON para activar/desactivar proveedores de scraping (TioPlus, FuegoCine, Supabase DB) y reordenar su prioridad de visualización.

GET /api/v1/search?q=avatar

Búsqueda General: Busca todo tipo de contenidos (Películas, Series, Animes y Doramas) sobre el catálogo indexado con coincidencia por substring (ej. ma encuentra Matilda). Respuesta lean por defecto (sin cast ni reproductores) para máxima velocidad — el detalle se obtiene en /api/v1/media/{id}; usa ?full=true para el objeto completo. Reporta total_results como el total real del catálogo, con has_more y next_page para scroll infinito.

GET /api/v1/openapi.json

Especificación completa OpenAPI 3.0.3 en formato JSON para lectura de máquinas, validación de esquemas y generación de clientes.

GET /api/v1/feeds/home?country=CL

Obtiene el Feed principal de inicio con carruseles horizontales estilo Netflix y banners destacados.

GET /api/v1/media/scary-movie-6

Obtiene los metadatos completos y reproductores de un título específico por su slug o ID.
